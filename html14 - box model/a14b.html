Tags de agrupamento São essenciais para a organização do layout no html. Embora não haja diferença no que diz respeito à funcionalidade das tags, deve-se utilizá-las corretamente para fins de organização e para melhorar a visibilidade externa do site por mecanismos de busca.

Descrição das tags de agrupamento

Essas tags são puramente semânticas, logo são funcionalmente idênticas. No entanto, devem ser utilizadas de acordo com a sua finalidade.

A tag <article> Comumente contém textos pequenos de assuntos relacionados, artigos.

Tag usada para textos "alternativos", como legendas.

Outras Grouping Tags

  • <header>, <main> e <footer> São as maiores estruturas do layout: cabeçálho, corpo e rodapé, respectivamente;
    Obs.: Não confundir <header> com <head>
  • A tag <nav> é utilizada para menus de navegação;
  • A tag <section> separa o conteúdo em "seções" conforme necessário;
